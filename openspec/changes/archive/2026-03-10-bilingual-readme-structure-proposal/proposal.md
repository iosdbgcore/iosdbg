## Why

项目已有完整的技术规范文档，但缺少面向用户的中英双语 README，导致新用户难以快速理解项目用途、安装方法和使用方式。添加结构化的双语 README 可以降低使用门槛，提升项目可访问性。

## What Changes

- 在项目根目录创建 `README.md`（中文）和 `README.en.md`（英文）
- 包含项目简介、功能特性、安装指南、快速开始、使用示例
- 提供清晰的项目架构说明和贡献指南
- 链接到现有的 OpenSpec 技术文档

## Capabilities

### New Capabilities
- `bilingual-readme`: 中英双语 README 文档生成能力，包含项目介绍、安装、使用、贡献等标准章节

### Modified Capabilities
<!-- 无现有能力需要修改 -->

## Impact

- 新增文件：`README.md`（中文）、`README.en.md`（英文）
- 不影响现有代码或技术文档
- 提升项目对中英文用户的友好度
