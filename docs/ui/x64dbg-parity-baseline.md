# x64dbg 同构 UI 基线清单

本清单用于实现任务 2.1，定义与 x64dbg 保持一致的最低交付基线。

## 1) 窗口布局

- 左侧主区域：反汇编视图（默认宽度约 62%）
- 右上区域：寄存器面板（默认高度约 43%）
- 右下区域：内存查看器
- 顶部区域：控制面板（Open/Attach/Step/Continue）
- 底部区域：状态栏（附加状态、目标、错误分类）

## 2) 面板分区

- 反汇编区：地址 + 指令 + 操作数 + 断点标记
- 寄存器区：名称 + 十六进制值
- 内存区：地址 + 十六进制 + ASCII
- 控制区：加载、附加、执行控制按钮

## 3) 核心交互

- 支持按 PID/进程名附加目标进程
- 附加成功后可直接使用 Step/Continue/断点操作
- 断点点击切换（空心/实心）并在执行命中时反馈状态
- 附加失败时展示稳定错误分类（permission_denied/target_not_found/timeout/lldb_error）

## 4) 状态样式

- 正常：中性色文字
- 附加中：橙色提示
- 附加成功：绿色提示
- 附加失败：红色提示
- 当前指令高亮：深色背景 + 蓝/橙强调色

## 5) 验收口径

- 保证“信息架构 + 交互路径 + 状态反馈”一致
- 不要求像素级逐点一致
- 若存在实现差异，必须在 `docs/ui/x64dbg-reuse-feasibility.md` 记录原因
